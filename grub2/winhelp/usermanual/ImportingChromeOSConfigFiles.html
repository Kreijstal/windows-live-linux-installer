Importing ChromeOS Config Files


Note: This procedure is experimental.

Support must be obtained from the Brunch and ChromeOS forums.



This facility lets you import GNU Grub version 2 menu entries from a configuration file generated by Brunch



Here are the steps to import ChromeOS menu entries:



1.        Download and install the Brunch package as shown the YouTube tutorial available from Tricknology here



                       https://www.youtube.com/watch?v=OicSb2QpAVk


                               


2.        Brunch will create a configuration file called chromeos.img.grub.txt



3.        Boot into Windows and run Grub2Win. Click on the yellow "Manage Boot Menu" button.



4.        Once the Menu Configuration screen comes up click the blue "Chrome" button at the top of the screen.



5.        When the file dialog appears select the chromeos.img.grub.txt file from step 2 above.



6.        You will then see a selection screen containing the Chrome and Brunch menu items available for import.







Click the checkbox for the menu items you want to import and click "Import Selected Items.



The menu items you imported will now appear at the bottom of the Grub2Win menus as custom code.

You can now click the imported items to edit the code or move them up or down in the boot menu.







Note:                Special characters


 The following special characters are not allowed in menu titles      \   /   :   *   ?   $   &   "   >   <   |  '   }   {  

 The import function will remove any invalid characters.
