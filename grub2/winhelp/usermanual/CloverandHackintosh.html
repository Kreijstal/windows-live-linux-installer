Clover and Hackintosh




Clover is a standalone EFI module used to boot OSX Hackintosh systems. Grub2Win supports Clover on 64 Bit EFI computers.



To install Clover, simply add a bootmenu entry by using the "Add A New Entry" button on the Manage Boot Menu screen.

Select "clover" as the type when you are prompted.



Once Clover is installed, Grub2Win will automatically manage the Clover modules in your EFI partition.

This happens when you add or delete a Clover type menu entry. Also when a new level of Clover is released.


Grub2Win will then download, update and delete Clover modules as needed.





The Clover software documentation is here:



https://sourceforge.net/projects/cloverefiboot/




Setting up Clover to boot OSX can be very complex.

Most Clover boot options are contained in the /efi/CLOVER/config.plist file.

The initial file supplied when you install Clover with Grub2Win is basic, simple and generic.


You can click the "EFI Partition Operations" button on the main screen.

Then use the EFI browse function to edit the config.plist file as required.



The definitive guide to setting up a Hackintosh PC using Clover can be found on the TonyMac X86 website here:



https://www.tonymacx86.com/
