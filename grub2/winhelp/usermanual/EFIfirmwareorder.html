EFI Firmware Order

(EFI mode only)







Use the up / down arrows on the right to change the EFI firmware sequence




This screen will appear when you click the EFI Firmware Boot Order button on the Main Configuration screen.


This is a convenient way to change the boot order in your machine's EFI firmware. If necessary, you can also change the order by going into the EFI setup for your motherboard. Consult your motherboard documentation for details.



"Set Grub2Win As Default" button, Grub2Win will be moved to the top of the list and will become your default EFI boot manager program. You will be asked to confirm this change and it will be processed immediately. This button only appears if Grub2Win is not already the default boot manager.



"Set Windows As Default" button, Windows will be moved to the top of the list and will become your default EFI boot manager program. You will be asked to confirm this change and it will be processed immediately. This button only appears if Windows is not already the default boot manager.



When you change the boot order, an  "Apply Updates" button will appear. Once you are satisfied with your changes, click this button to process them. You can also cancel any pending updates by clicking the "Cancel Pending Updates" button.



Once any updates successfully complete, you can return to the  Main Configuration  screen.




Note: 


Some EFI firmware refuses to honor the boot order as specified in EFI standards. This problem often occurs in older firmware releases, or if the EFI pointers in NVRAM are corrupted. You can force your EFI firmware to load Grub2Win by checking the "Force Unconditional Load Of Grub2Win" box. This will override the normal EFI boot order.
