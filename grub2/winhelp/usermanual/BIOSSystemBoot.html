BIOS System Boot



For systems with BIOS firmware, the first screen you see at boot time is


 Windows Boot Manager




If you want to start Windows, you must select it here in the Microsoft Windows Boot Manager screen.



If you want to use the Grub2Win boot menu, select "Grub 2 For Windows".


Grub2Win cannot directly load Windows on systems with BIOS firmware.
