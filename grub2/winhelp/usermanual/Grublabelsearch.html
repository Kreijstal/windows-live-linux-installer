Configuring Grub2 For Partition Label Search



     You can use Grub2Win to search for a partition by its label at boot time.


First, set the partition label using these instructions: Setting Your Labels.


Now that the label is set, simply run Grub2Win and edit the OS details for the menu entry.

Choose Load The Kernel By   Partition Label   as shown below.

Now select the partition you want to load at boot time.


In this example, we are searching for a Ubuntu Linux partition with a label of ubuntu-22.04

Grub2Win will generate a grub2.cfg file entry with the proper code to locate the partition at boot time.
