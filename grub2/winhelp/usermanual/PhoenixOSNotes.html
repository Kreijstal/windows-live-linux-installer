PhoenixOS Notes



Grub2Win supports PhoenixOS on 64 Bit Windows PC's. You can download PhoenixOS for Windows here:


http://www.phoenixos.com/en/download_x86


Once you have Phoenix installed, you can use Grub2Win to boot it. You need to know the name of the Phoenix kernel file to set this up.


For the current release, the kernel file name is /PhoenixOS/kernel.


If you have trouble finding the directory name, you can also use the Grub2Win chainloader to boot the partition directly by entering the disk drive address.


Make sure to install the 64 bit version of the PhoenixOS distribution. Otherwise PhoenixOS will not boot.


For Nvidia support, check the Nvidia support box on the Editing OS Details screen.


PhoenixOS is very sensitive to your specific hardware. Many issues can be remedied by changing boot parms. The default parms that Grub2Win inserts when you add a PhoenixOS system should work in most cases. You can change the parms if desired.
