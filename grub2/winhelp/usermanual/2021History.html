Version 2.3.1.5        November 8, 2021




Grub2Win setup now checks to see if the installed release is old.

Setup then checks for old incompatible configuration files.


If the old release is not compatible, a warning message will appear.

With user approval, a clean install of Grub2Win will then be performed.


The old incompatible C:\grub2 directory will be saved as C:\grub2.archive.


Configuration files from recent Grub2Win releases will still be migrated.

Fixed some errors in the import scan routines.

Version 2.3.1.1        October 4, 2021




Grub2Win now supports Windows 11.


Minor enhancements to the GUI and help files.






Version 2.3.1.0        September 23, 2021




Added limited support for EFI partitions on USB flash drives.

The drive must be MBR style with a letter assigned.

The partition type in the MBR must be “EF” per standards.


See the Grub2Win help topic “EFI Partition Info and Tips”.






Version 2.3.0.9        September 22, 2021




Added a warning if no OS boot menu entries have been defined.


Updates to the getpartition function help.


Note: Your Grub2Win EFI partition modules will be refreshed to level 150.






Version 2.3.0.8        August 25, 2021




Minor enhancements to the GUI and help files.






Version 2.3.0.7        August 20, 2021




Allow Grub2Win to run on Windows 11 beta systems.


Windows 11 is not yet supported by the AutoIt compiler.

Grub2Win is written in the AutoIt language.


A warning message will be issued if Windows 11 is detected

at setup time. Severe errors may occur if the install is allowed.


The user is given the choice to accept the risk of errors and

continue the install, or to cancel the installation.

Version 2.3.0.5        July 29, 2021




****  This release changes the format of certain Grub2Win .cfg files.

****  If you revert to an earlier release, you must restore all of

****  your .cfg files from the earlier release.  

****

****  The easiest way to do this is to simply restore the C:\grub2

Version 2.3.0.0        June 9, 2021




***  This is a major release  ***


Grub2Win now boots and contains the new GNU Grub 2.06 kernel and libraries.


-------------------------------------------------------------------------------------------------


The following info is from the GNU Grub news file:


New in 2.06:


* GCC 10 support.

* clang 10 support.

* SBAT support.

* LUKS2 support.

* Drop small MBR gap support.

* Xen Security Modules (XSM/FLASK) support.

* The lockdown mechanism similar to the Linux kernel one.

* Disable the os-prober by default.

* Many backports of GRUB distros specific patches.

* BootHole and BootHole2 fixes.

* ...and tons of other fixes and cleanups...
