POSROG Install Instructions

                                        POSROG



=============================================================================================================


       *** Remember here only includes 3 kernel variants which we have adjusted ***


=============================================================================================================


How to install POSROG V3U8 using Grub2Win :



1. Download and install the latest version of Grub2Win from SourceForge using this link.

    You must be at Grub2Win version 2.3.7.3 build 1613 or higher.


       https://sourceforge.net/projects/grub2win/



2. Run Grub2Win and click the yellow "Manage Boot Menu" button on the main screen.


    Then click the orange "Add A New Entry" button on the Menu Configuration screen.


    Click the "Type" pulldown and select  posrog


    Follow the instructions to complete the Grub2Win POSROG configuration. Grub2Win will restart.


    Now continue with step 3 below to set up the POSROG Linux partition.



******************************************************************************************************************************************



3. Prepare a new partition with format ext4 using disk genius software or other software (this is recommended for clean install mode)


4. Create a folder with name "posrog"


5. Create a folder with name "data" inside "posrog" folder (remember clean install mode requires an empty data folder)


6. Extract or take posrog files from ISO using 7-zip by right clicking open archive or other software and Copy file and move it to "posrog" folder:


               - initrd.img

               - kernel-empat        <====> kernel 5.4

               - kernel-lima        <====> kernel 5.8

               - kernel-enam        <====> kernel 5.10

               - posrog

               - ramdisk.img

               - system.img


7. Restart and boot into grub2win (remember disable Secure Boot for UEFI), select kernel for first boot (for AMD Ryzen with IGPU select kernel 5.4 , 5.8 or 5.10)


8. Wait for booting process to enter the setup wizard (not added for the SLE version) select "Set as a new" then don't connect to wifi or network first then select "do not use network for setup" and follow next steps, if you experience crash in the setup wizard you can disable via terminal press ALT+F1 type pm disable com.google.android.setupwizard
