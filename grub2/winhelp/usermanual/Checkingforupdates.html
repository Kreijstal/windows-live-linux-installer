Checking for updates

Using the Grub Menu ››

Check For Updates





This screen will appear when you click the Updates button on the Main Configuration screen.


This reminder screen will first appear automatically, 30 days after the Grub2Win release date. You can change the automatic reminder schedule to 60 or 90 days. You can also disable the reminders by un-checking the box shown above.


If you click the Check For Updates Now button, Grub2Win will check SourceForge to see if there is a newer release.





You will then see this screen







You will have options to View The Change Log or Visit The Official Grub2Win Site. 


If you are not running the current release, you will receive a warning and an additional button will appear in green:


       Upgrade to Grub2Win Version x.x.x.x


If you want to update to the latest version, click the green button. The latest release of Grub2Win will then be downloaded and extracted.



If you are already running the latest release, the Refresh Grub2Win Version x.x.x.x button will appear.

You can click it to reinstall the latest version and build.



You will then be taken to the Grub2Win Setup Screen to install the release.
